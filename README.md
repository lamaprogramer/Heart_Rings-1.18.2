# Heart_Rings
This mod adds heart rings, which give extra hearts, and have a unique ability when you use them.

There are 5 rings: Red, Blue, Green, Yellow, and Purple. Red is the weakest, whilst Purple is the strongest.


# Requirements

Fabric API

Trinkets
